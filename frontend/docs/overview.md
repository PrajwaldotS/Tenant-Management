# Frontend Overview

This is a role-based admin dashboard for a Property & Tenant Management System.

## Tech Stack
- Next.js (App Router)
- TypeScript
- TailwindCSS
- shadcn/ui
- Zustand (state management)
- Axios (API calls)
- Zod (form validation)

## Core Features
- Authentication (JWT-based)
- Role-based UI rendering
- Dashboard analytics
- CRUD management (Users, Properties, Tenants)
- Payment tracking

## Design Principles
- Clean, minimal UI
- Reusable components
- Server-first rendering where possible
- Client components only when necessary
